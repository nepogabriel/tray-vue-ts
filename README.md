## Sobre
Aplicação responsáve por consumir a API SALES AND SELLERS.

## Tecnologias utilizadas
- Typescript
- Vue 3
- Axios
- typescript-cookie

# Rodando projeto
## Pré-requisitos
- Git
- Docker

## Portas necessárias
- Porta 5175

## Passo a Passo com DOCKER
- 1- Clonar
```URL
https://github.com/nepogabriel/tray-vue-ts.git
```
- 2- Entrar na pasta
```
cd tray-vue-ts
```

- 3- Construir imagem
```
docker composer build
```

- 4- Executar imagem
```
docker composer up -d
```

- 5- Links de acesso
```URL
http://localhost:5175/
```

## Passo a Passo sem DOCKER
- 1- Clonar
```URL
https://github.com/nepogabriel/tray-vue-ts.git
```
- 2- Entrar na pasta
```
cd tray-vue-ts
```

- 3- Construir imagem
```
npm i
```

- 4- Executar imagem
```
npm run dev
```

- 5- Links de acesso
```URL
http://localhost:5174/
```
